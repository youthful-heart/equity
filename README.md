# 小米 DDX 每日对比 📊

> 零成本 · GitHub Actions 自动运行 · 每日推送

基于「零成本 AI 股票分析」知识库的 GitHub Actions 路径，专为**小米集团（01810.HK）**打造的 **DDX 大单动向每日对比**仓库。

Fork 即用，每天收盘后自动抓取数据、生成对比报告，全程零费用。

---

## 🚀 功能

| 功能 | 说明 |
|------|------|
| 📈 **股价+K线数据** | 日线行情（开盘/收盘/最高/最低/成交量/成交额） |
| 💰 **资金流向数据** | 主力净流入/超大单/大单/中单/小单 |
| 📊 **DDX 指标计算** | 大单动向（DDX）+ 累计 DDX + 对比趋势 |
| 📉 **多周期对比** | 当日 / 3日 / 5日 / 10日 / 20日 对比 |
| 🖼️ **趋势图** | DDX 柱状图 + 股价走势叠加图 |
| 📋 **Markdown 报告** | 自动生成可读报告，推送到 GitHub |
| 🌐 **GitHub Pages** | 可选部署，浏览器直接查看历史报告 |

## 🛠️ 一键部署（5 分钟）

### 第一步：Fork 本仓库

点击右上角 **Fork** 按钮，将仓库复制到你的 GitHub 账号。

### 第二步：启用 GitHub Actions

进入你的仓库 → **Actions** 选项卡 → 点击 **"I understand my workflows, go ahead and enable them"**。

### 第三步：配置 Secrets（可选）

默认情况下，**无需任何配置即可运行**，数据全部来自免费公开接口。

如需自定义通知渠道，可在 **Settings → Secrets and variables → Actions** 添加：

| Secret | 说明 | 必填 |
|--------|------|:----:|
| `STOCK_CODE` | 股票代码，默认 `01810`（小米） | ❌ |
| `MARKET` | 市场，默认 `HK`（港股） | ❌ |
| `WECHAT_WEBHOOK` | 企业微信机器人 Webhook | ❌ |
| `FEISHU_WEBHOOK` | 飞书机器人 Webhook | ❌ |
| `EMAIL_SENDER` | 发件邮箱 | ❌ |
| `EMAIL_PASSWORD` | 邮箱密码/应用密码 | ❌ |
| `EMAIL_RECEIVERS` | 收件邮箱（逗号分隔） | ❌ |

### 第四步：手动触发一次测试

进入 **Actions** → **每日 DDX 对比** → **Run workflow** → 等待 2 分钟，查看 `output/` 目录下的报告。

### 第五步：查看报告

每次运行后，报告会自动提交到仓库的 `output/` 目录：

```
output/
├── ddx_report_2026-07-10.md    ← 每日对比报告
└── ddx_chart_2026-07-10.png    ← DDX 趋势图
```

## 🔧 自定义

### 修改股票

编辑 `.github/workflows/daily_ddx.yml`，修改 `STOCK_CODE` 环境变量：

```yaml
env:
  STOCK_CODE: '00700'   # 改为腾讯
  MARKET: 'HK'
  STOCK_NAME: '腾讯控股'
```

### 修改运行时间

修改 workflow 中的 cron 表达式：

```yaml
on:
  schedule:
    - cron: '30 8 * * 1-5'   # 北京时间 16:30 运行
```

港股交易时间：9:30-16:00，建议 16:30 后运行（UTC 08:30）

### 添加更多对比股票

修改 `scripts/config.py` 中的 `STOCKS` 列表即可。
